# 싸피북 (SSAFYBook) - E-Book 도서관 DB 프로젝트

## 1. 프로젝트 개요

본 프로젝트는 **E-Book 도서관 웹 애플리케이션 '싸피북(SSAFYBook)'**의 데이터베이스를 설계하고 구축합니다.

주요 목표는 사용자의 E-Book 대출/반납을 관리하고, 독서록 및 도서 추천 기능을 지원하는 확장 가능한 DB 스키마를 만드는 것입니다. E-Book의 특성을 반영하여 '무제한 동시 대출'을 기준으로 하되, '사용자별 N권'까지 대출을 제한하는 정책을 적용했습니다.

## 2. 요구사항 기반 주요 기능

DB 스키마는 다음과 같은 핵심 기능들을 지원하도록 설계되었습니다.

### 📚 기본 기능

* **회원 관리:** 회원 가입, 조회, 수정, 삭제, 로그인/로그아웃 기능
* **도서 관리 (E-Book):** E-Book 도서 정보(저자, 출판사, 카테고리 포함) 등록, 검색, 수정, 삭제
* **대출 및 반납:** E-Book 대출 신청, 반납 처리 (사용자별 N권 대출 제한 로직 포함)
* **독서록:** 도서별 독서록 작성, 별점 평가, 조회, 수정, 삭제 기능

### ✨ 추가 기능

* **독서 마라톤:** 회원이 읽은 (독서록을 작성한) 도서의 `page_count`를 합산하여 통계를 내는 기능
* **도서 추천:** 사용자의 독서 기록(대출, 독서록, 별점) 패턴을 분석하여 맞춤형 도서를 제공하는 기능
